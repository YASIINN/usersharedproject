# Öğrenci Paylaşım App Demo Frontend VueJS

## `npm install`
diyerek bağımlılıkların yüklenmesi

## `npm run serve`
diyerek projenin ayağa kaldırılması

## Katmanlar
- Assets (Uygulama Sabit Dosyaları )
- Components 
- Provider
- Service (Api Requests)
- Plugins   
- Router 
- Store (Vuex Modules)

2. Axios

5. Vuex
6. Underscore

9. Underscore
